# Indian Weather Data Analysis and Visualization (1901–2017)

## Project Overview
This project analyzes and visualizes historical temperature data of India from 1901 to 2017. The dataset includes monthly average temperatures per year, enabling exploration of long-term climate trends and seasonal variations.

## Features
- Data loading and cleaning using `pandas`
- Calculation of yearly and monthly average temperatures
- Visualizations including line charts, bar charts, and heatmaps with `matplotlib` and `seaborn`
- Identification of the hottest and coldest years
- Modular scripts for focused analysis and visualization
